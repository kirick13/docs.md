Nvigation categories

Each category contains links or another categories.

Properties

title
string

Text of link or category.

url
string?

Link's URL.

content
array?

Content of category.

Examples

Example configuration only navigation's block properties
{
  "content": [
    {
      "title": "Main page",
      "url": "/"
    },
    {
      "title": "Links",
      "content": [
        {
          "title": "About",
          "url": "/about"
        },
        {
          "title": "Contacts",
          "url": "/contacts"
        }
      ]
    }
  ],
}
