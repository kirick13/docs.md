Variables

That paramaters are located inside vars property. By changing them you will customize texts and titles on your website.

Property aside

info
string? = null

Title of <aside info> element. If null, that block will have no title.

warning
string? = "WARNING"

Title of <aside warning> element. If null, that block will have no title.

Property parameters

required
string = "required"

Text on required parameter badge.

deprecated
string = "deprecated"

Text on deprecated parameter badge.

experimental
string = "experimental"

Text on experimental parameter badge.

Property code_example

title_request
string = "Request"

Default title of Request code example code example.

Examples

Default configuration only vars properties
{
  "aside": {
    "info": null,
    "warning": "WARNING"
  },
  "parameters": {
    "required": "required",
    "deprecated": "deprecated",
    "experimental": "experimental"
  },
  "code_example": {
    "title_request": "Request"
  }
}
