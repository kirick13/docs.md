
# Navigation blocks

Each block contains a list of links or categories.

Blosks are located inside `navigation` array.

## Properties

<<< PARAMETERS
title: string?
Block title.
<<<
content: array
Array of links or categories.
<<<

# MARK: Examples

## Example configuration

only `navigation` properties

```json
[

  },

  }
]
```
